# Dependencias  
Certifique-se de ter instalado na sua máquina o Node.js.  
Ele pode ser instalado nesse site:  
[Download(Node.js)](https://nodejs.org/en/download)  
  
Tenha instalado também o yarn.  
Ele pode ser instaldo nesse site:  
[Download(yarn)](https://classic.yarnpkg.com/lang/en/docs/install/#windows-stable)

É necessário ter instalado no seu dispositivo mobile o aplicativo expo.  
Ele pode ser instalado na Play Store.   
![alt text](./src/assets/image.png)
# Instalação
Após baixar o repositório, abra o terminal e instale as dependencias do projeto com o seguinte comando:  
` yarn `  
# Execultar
Para execultar o projeto precisará operar o seguinte comando:  
` yarn start `  
Aparecerá um QR Code no terminal, depois é só apontar a câmera do aplicativo instalado expo para o QR Code que o projeto execultará no seu dispositivo mobile.
